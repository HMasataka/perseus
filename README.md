# perseus

Go slice operation extension

## Supported Function  

 * In
 * Index
 * Shift
 * Unshift
 * Delete
 * Cut
 * Insert
 * InsertVector
 * Pop
 * Reversed
 * Extend
 
## Supported Type  

 * int
 * int8
 * int16
 * int32
 * int64
 * uint
 * uint8
 * uint16
 * uint32
 * uint64
 * float32
 * float64
